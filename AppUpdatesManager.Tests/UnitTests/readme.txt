UnitTests: Содержит модульные тесты для отдельных компонентов системы, разделенных по слоям и функциональности.

Domain: Тесты для доменных сущностей и объектов-значений, проверяющие бизнес-логику на уровне модели.
Application: Тесты для команд, запросов и сервисов слоя приложения, проверяющие логику приложения без взаимодействия с инфраструктурой.
Infrastructure: Тесты для репозиториев и внешних сервисов, возможно с использованием моков или фейков.
